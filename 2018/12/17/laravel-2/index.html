部署到heroku上

heroku login
进入项目文件夹
echo web: vendor/bin/heroku-php-apache2 public/ > Procfile //这个出现问题一般会went wrong
heroku create name
heroku buildpacks:set heroku/php //这个是声明使用的语言，我这里用的是PHP
设置app key
php artisan key:generate
heroku config:set APP_KEY=<your_app_key>
git add -A
git commit -m “”
git push heroku master
heroku open

数据库迁移

php artisan migrate
运行迁移，up被调用
回滚迁移，down被调用

up方法中调用Schema类的create方法创建users表

create 方法 接受两个参数
1.数据表名称
2.$table的闭包

down逆向操作 drop 删除表

Schema::dropIfExists('users');
