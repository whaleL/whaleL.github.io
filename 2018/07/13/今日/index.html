#　刑法

  • 判处死刑会器质性德，在死刑缓期执行期间，如果没有故意犯罪的话，二年期满后，减为无期徒刑。如果有重大立功表现德，减为25年有期徒刑。
    故意犯罪的话，报请最高人民法院核准后执行死刑。
    故意犯罪未执行死刑的，死刑缓期执行的期间重新计算，并报请最高人民法院进行备案。
  • 累犯，不满十八周岁跟非故意犯罪 五年内不算。其他的五年内构成累犯。
    无期徒刑的人会被剥夺终身政治权利。
